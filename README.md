Sakuryn

Sakuryn is a lightweight desktop music player built as a learning-focused project.
It allows users to load and play local audio files while exploring core concepts in desktop application development.

The project focuses on simplicity, clean structure, and gradual feature expansion.

Features:

Load local music files (MP3)
Play, pause, and navigate tracks
Basic music library handling
Minimal user interface


Tech Stack:

JavaScript
HTML
CSS
Electron

Purpose:

Sakuryn was created to:
Practice JavaScript and desktop application development
Learn file system access and audio playback
Build a foundation for future features such as playlists and settings


Project Status:

This project is currently under development.
New features and improvements are added incrementally.

Planned Improvements:

Persistent music library
Playlist support
Settings panel
UI and usability improvements
